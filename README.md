<h3></h3>

Sample Python application on Django with PostgreSQL database.

<h3>Requirements</h3>

____


- django 4.0.1
- Pillow 9.0.0
- psycopg2-binary 2.9.3
- django-prometheus 2.2.0

<h3>Deployment</h3>

____



- install Docker
- install Docker Compose

* Build:
```shell
docker-compose build
```

* start application:
```shell
docker-compose up -d
```


* stop application:
```shell
docker-compose down
